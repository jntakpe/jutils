Bienvenue sur l'application jUtils !

Cette application donne l'accès à différents outils notament un annuaire collaborateur/item et item/collaborateur ainsi qu'un module de commande de cafés Nespresso.

Le code de l'application est disponnible sur Github à cette adresse : jUtils, vous pouvez y repoter les bugs ou y proposer des évolutions.

Les technos utilisées par jUtils sont :

  • Java EE
  • Spring framework (MVC + security + data)
  • Hibernate
  • Thymeleaf
  • AngularJS
  • Bootstrap
